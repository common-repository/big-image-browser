Full screen view of images that is in topic page. Keys navigated.

== Description ==

Show big (original) images from blogs post page. It get all link hrefs with image anchors
(like <a href="some_big_image.jpg"><img src="its_small_copy.jpg"></a>) and then leafs through with big images in full screen.
It allows simple key (left, right, esc) and mouse navigating.

== Installation ==

1. Upload `big-image-browser` directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to any page your blog with hyperliked image and click it
